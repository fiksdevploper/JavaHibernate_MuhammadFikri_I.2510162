# JavaHibernate_MuhammadFikri_I.2510162

Proyek ini merupakan implementasi **CRUD (Create, Read, Update, Delete)** menggunakan **Java Hibernate** dengan integrasi ke database MySQL. Dibuat sebagai bagian dari pembelajaran OOP, GUI, dan database integration.

---

## ✨ Fitur Utama
- Implementasi CRUD dengan Hibernate ORM
- Koneksi ke database MySQL
- Validasi input sederhana
- Struktur OOP yang modular
- GUI berbasis Java Swing

---

## 📂 Struktur Proyek
- `src/` → berisi source code Java
- `entity/` → class model untuk tabel database
- `dao/` → Data Access Object (CRUD logic)
- `util/` → konfigurasi Hibernate
- `resources/` → file konfigurasi (hibernate.cfg.xml)

---

## ⚙️ Prasyarat
- Java JDK 8 atau lebih baru
- MySQL Server
- Hibernate ORM (library sudah disertakan di repo)
- IDE (NetBeans/IntelliJ/Eclipse)
